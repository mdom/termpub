# NAME

App::termpub - read epubs in the terminal

# SYNOPSIS

termpub _file_...

# DESCRIPTION

termpub is a _terminal_ viewer for epubs.

# KEY BINDINGS

- n

    Go to the next chapter.

- p

    Go to the previos chapter.

- KEY\_DOWN, SPACE

    Scroll down.

- KEY\_UP

    Scroll up.

- q

    Quit.
